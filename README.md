# Arabic Realtime Sign Detection ✋


A Python application for **real-time Arabic sign language detection** using hand tracking and a trained deep learning model. The system recognizes hand signs, predicts the corresponding Arabic letter, displays it on the screen, and can optionally speak the letter aloud.

---

## Features

- Real-time hand tracking using webcam input  
- AI model predicts Arabic sign letters from hand gestures  
- On-screen display of predicted letter  
- Optional audio output of the recognized letter  
- Easy to run locally with minimal setup
